Open-Source Arms Transfers Map (2000–2024)

A public-interest research tool for exploring global arms transfer trends over time.

What this is

This website provides an interactive world map for exploring arms transfer flows by year. You can view a selected country as an exporter, importer, or both, then drill down into partners and systems.

Important: Values represent SIPRI Trend-Indicator Values (TIV)—a standardized indicator of transfer volume, not a financial price.

Interpretation guide

  • As exporter shows flows from the selected country.
  • As importer shows flows to the selected country.
  • Both shows all flows involving the selected country.
